CONCLUSIONES:

 - Debido a un problema incierto no se pudo instalar express-generator ni crear la aplicacion de express
   tal como esta en el laboratorio. Se importó una carpeta con la app ya creada desde git.
 - La creacion de un servidor mediante express es similar a node, siendo mas intuitivo.
 - El navegador usado para probar los middlewares de nuestro servidor es Chrome.
   La extension usada es Postman.
 - Usando middlewares de express podemos identificar el tipo de peticion http y realizar la accion correspondiente,
   aún si es sobre una misma url.
 - Tambien podemos dar seguridad a nuestros directorios agregando un prefijo falso necesario para acceder a nuestros archivos en el servidor.
 - Los middlewares nos ayudan a controlar los errores que se puedan generar mientras se navega por nuestro servidor.
   Para evitar que el servidor se cierre y realizar medidas preventivas.
